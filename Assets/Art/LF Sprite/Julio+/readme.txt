1. Installation
----------------------------------------------------------------------

 Paste the two folders and the dll into your LF2 directory (ai, sprite
 and the ddraw.dll). For security reasons I advise you to always get
 the latest release version of the dll from the official source here:
 https://www.mediafire.com/folder/pxd026nqoqgw0/LF2ddWrapper
 You can also check out the source and compile it yourself.
 
 Then either replace the data.txt file inside the data
 folder with the included one or edit it like this:

 Add these four lines behind <object>:
  id:  55 type: 0 file: sprite\julio\julio.dat
  id:  79 type: 0 file: sprite\julio\julio+.dat
  id: 230 type: 3 file: sprite\julio\julio_ball.dat
  id: 413 type: 2 file: sprite\julio\julio_suit.dat

3. Uninstall
----------------------------------------------------------------------

 Simply rename the included data_backup.txt into data.txt and copy it
 to the data folder, overwriting the installed one, or rewind every
 step from the installation.
