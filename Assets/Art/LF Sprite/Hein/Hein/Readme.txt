Installation:
- Move all folders into your LF2-directory
- Create a folder ai and paste the file 70.as in it (you must have the "ddraw RELEASE" for the AI to work)
- Add these lines to data.txt (to the object-part):

id:  70  type: 0  file: data\hein.dat
id: 230  type: 3  file: data\hein_arrow.dat
id: 231  type: 3  file: data\hein_arrow2.dat
id: 232  type: 3  file: data\hein_stararrow.dat
