Installation:
Copy these lines and paste them into the data.txt in your LF2-folder:

id: 21  type: 0  file: data\darius.dat
id: 301  type: 3  file: data\darius_col.dat
id: 302  type: 3  file: data\darius_ball.dat
id: 303  type: 3  file: data\darius_blast.dat
id: 304  type: 3  file: data\darius_zap.dat
id: 305  type: 3  file: data\darius_crash.dat

Put the sprites in  the sprite\sys folder and the .dat files into the data folder.

Move list:

.D>A    Lightning Bolts
.D^A    Chasing Zapper
.DvA    Aerial Bang 
.D>J    Lightning Cannon
.D^J    Lightning Explosion
.DvJ    Detonation (MP-cheat only)
.DA     Self Heal
.DA+A   Darius Crash (Press attack during Self Heal)

Index:
D = Defend
J = Jump
A = Attack
^ = Up
v = Down
> = Left/right
